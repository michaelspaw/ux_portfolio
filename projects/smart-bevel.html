Smart Bevel

Advanced non-destructive beveling of edges for polygonal geometry

Role

Functionality advocate and Senior UX Designer

Timeline

2024 - 2025

Tools

3dsMax, Maya, Photoshop, Figma

Overview

Smart Bevel addresses long-standing issues with traditional modeling approaches to "relieving edges" or creating chamfers and bevels for polygonal modeling. This tool has been designed to be integrated into both 3dsMax and Maya as a replacement for legacy functionality.

The Ask

As a Modeler, I want to create an arbitrarily large Bevel with no limitations and no post cleanup.

The Problem

Creativity vs Constraints

  • Current tools can't easily produce the forms artists want to create
  • Geometrical constraints set hard limits
  • Time-consuming manual cleanup forces hard decisions
  • Artists turn to other tools to realize their vision

Competitive Analysis

Existing Alternatives

  • Modo Mesh Fusion
  • Blender MeshMachine Offset cut
  • Blender Bevel after Boolean
  • Maya/Max Hard Mesh
  • Max Smooth Boolean

Why These Options Aren't Widely Used

  • They only do part of the ask
  • Not interactive
  • Fiddly Setup
  • Reliability varies
  • Not Native

The Solution

How Can We Do Better?

  • Built in
  • Works with our existing tech
  • Better interactivity
  • Fewer limitations
  • More reliable
  • Consistent experience across DCCs

What We Need to Build

  • Consistently reliable results even with complex edges
  • Quick application, modification, and interactive evaluation
  • A replacement for Bevel/Chamfer
  • Convergence on Nomenclature and UX

Implementation Details

CutBack tech should replace the existing features for both Maya and Max:

  • Maya Bevel Tool
  • Max Chamfer Modifier

Future Development

  • Direct chamfers/bevels along Boolean intersection curves
  • Variable width radius
  • Custom curve-based profiles
  • Other applications: bridge, inset, beveled text
