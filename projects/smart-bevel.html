Smart Bevel

Advanced non destructive Beveling of edges for polygonal geometry

Role

Functionality advocate and Senior UX Designer

Timeline

2024 - 2025

Tools

3dsMax, Maya, Photoshop, Figma

Overview

Smart Bevel addresses long standing issues with traditional modeling approaches to "relieving edges" or creating chamfers and bevels for polygonal modeling. This tool has been designed to be integrated into both 3dsMax and Maya as a replacement for legacy functionality.

The Ask

As a Modeler, I want to create an arbitrarily large Bevel with no limitations and no post cleanup

The Problem

Creativity vs Constraints

  • Current tools can't easily produce the forms artists want to create
  • Geometrical constraints set hard limits
  • Time consuming manual cleanup forces hard decisions
  • Artists turn to other tools to realize their vision

Current Tools Limitations

Arbitrarily large filleted/chamfered transitions between features

Geometrical Constraints

An example of the current behavior in Max

  • Hard limits due to existing edges
  • Fails with the simplest of cases
  • More complex geometry can make the situation a real rat's nest

Manual Cleanup Challenges

  • The artist either needs to limit the size of the chamfer or
  • Geometry needs to be designed to accommodate such features
  • Last resort: Manual cleanup
