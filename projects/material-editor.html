Material Editor

Advanced material creation and management system

Role

UX Designer

Timeline

2019 - 2020

Tools

Figma, Substance Designer, Unity

Overview

The Material Editor is a comprehensive tool for creating, editing, and managing PBR materials in real-time. It provides artists with an intuitive interface for crafting complex material networks while maintaining performance and visual quality.

The Problem

Material creation in 3D applications often requires switching between multiple tools and dealing with complex node-based interfaces. Artists needed a more streamlined workflow for creating and managing materials across different projects.

The Work

I designed a material editor that features:

  • Visual node-based material graph
  • Real-time material preview
  • Preset library and sharing system
  • Cross-application material export
  • Performance optimization tools
