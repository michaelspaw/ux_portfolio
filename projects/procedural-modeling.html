Procedural Modeling

AI-powered 3D model generation system

Role

Lead UX Designer

Timeline

2020 - 2021

Tools

Figma, Python, TensorFlow

Overview

Procedural Modeling is an innovative system that uses artificial intelligence to generate 3D models based on user input and reference images. This tool revolutionizes the 3D modeling workflow by automating the creation of complex geometries while maintaining artistic control.

The Problem

Traditional 3D modeling is time-consuming and requires significant technical expertise. Artists needed a way to quickly generate high-quality 3D models while maintaining creative control over the final output.

The Work

I designed a user-friendly interface for the procedural modeling system that includes:

  • AI-powered model generation from sketches
  • Real-time parameter adjustment
  • Style transfer capabilities
  • Export options for various 3D formats
